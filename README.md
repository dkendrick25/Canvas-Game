#Yoshi Canvas game

###Overview of the Project
Welcome to Yoshi's World. Try and catch the egg but watch out for Boo! Try and beat your personal high score. Your score will drop to zero everytime Boo catches you.

##Technologies Used
* HTML Canvas
* JavaScript
